## Writing style rules

1. Prefer concrete verbs over buzzwords.
2. Avoid unverifiable claims (e.g., “best-in-class”, “expert in everything”).
3. Use evidence where possible (metrics, before/after, incident reduction, lead-time change).
4. Keep tone professional, clear, and direct.
5. If data is unavailable, describe scope and constraints honestly.
